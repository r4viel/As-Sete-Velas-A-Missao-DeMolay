# 🕯️ As Sete Velas: A Missão DeMolay

Um jogo de aventura em **Pixel Art** desenvolvido em Python para explorar a história e os valores da Ordem DeMolay.

## 🕹️ O Jogo

As Sete Velas: A Missão DeMolay, não é um jogo de luta sobre violência gratuita, mas sim sobre o conflito ideológico e a provação do caráter.
Na lore do jogo, o Templo Maçônico não é apenas um local físico, mas um nexo atemporal. Nele, as Sete Virtudes Cardeais da Ordem DeMolay (Amor Filial, Reverência, Cortesia, Companheirismo, Fidelidade, Pureza e Patriotismo) manifestam-se como desafios de combate.
Figuras históricas cruciais como (Jacques Demolay, godfrey de gonneville, hugues de peralde) ligadas diretamente à Ordem e que moldaram os conceitos de virtude e honra através dos séculos são invocadas neste nexo, e (Papa Clemente V, Rei Filipe o belo e seus soldados). Eles devem lutar entre si não para destruir o oponente, mas para provar a supremacia de sua própria convicção e pureza de intenção. O combate é o ritual; a vitória é a validação moral.

## 🛠️ Tecnologias
* Python 3.12
* Pygame
* Sys
* Os
* Pixel Art Studio

## 🚀 Como jogar
1. Certifique-se de ter o Python instalado.
2. Rode no terminal `pip install pygame`.
3. Execute `python DeMolayQuest.py`.
